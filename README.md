# master-thesis-serious-game

This repository contains the serious game developed as the artifact for my master's thesis.

The game is a browser-based learning intervention about cognitive accessibility barriers in digital interfaces. It is played in Norwegian Bokmål and follows six review cases for the fictional financial-services platform Kronkrøll Bank & Forsikring.

## Run locally

```sh
pnpm install
pnpm dev
```

For a production build:

```sh
pnpm build
```
